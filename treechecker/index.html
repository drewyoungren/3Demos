Tree Checker

A directed graph or digraph consists of two sets:

  • $V$, the set of vertices (or nodes), and
  • $E$, the set of edges, ordered pairs $(v_1, v_2)$ of nodes.

One way to store a digraph is as a Javascript object, where each node is a key and its value is the set of its children. For instance the digraph $$(V,E) = (\{a,b,c,d\}, \{(a,b), (a,c),(b,c), (d, b), (a,d)\})$$ would be 


{
    a: ["b","c","d"],
    b: ["c"],
    c: [],
    d: ["b"],
}
whereas a human would just draw it.

  • true if the graph has no cycles (paths that start and end at the same node), and is thus a tree, and
  • false if it does have a cycle.
